# 宠物狗年龄识别系统
基于注意力机制的宠物狗年龄智能识别系统，融合计算机视觉与自然语言处理技术，实现狗龄精准识别与个性化护理指南自动生成。

## 项目简介
本系统以宠物狗图像为输入，通过YOLO11完成狗头区域精准裁剪，基于ResNet50融合SE（Squeeze-and-Excitation）注意力机制构建年龄识别模型，结合Ollama本地部署的DeepSeek大模型，根据识别结果自动生成个性化护理指南，最终搭建集“图像识别-年龄判定-智能建议”于一体的CV+NLP融合服务系统。

## 环境依赖
推荐使用Python 3.8+，核心依赖库如下（附关键版本）：
torch==2.7.1+cu118 # 深度学习模型构建
torchvision==0.22.1+cu118 # 图像预处理、模型加载
numpy==1.26.4 # 数值计算与特征处理
opencv-contrib-python==4.11.0.86 # 图像检测、裁剪
scikit-learn==1.7.1 # 数据划分、模型评估
matplotlib==3.10.6 # 结果可视化、特征分析
pandas==2.3.2 # 数据集管理
PySide6==6.9.2 # 可视化 UI 开发
pillow==11.0.0 # 图像读写与预处理
tqdm==4.67.1 # 训练进度可视化
scipy==1.15.3 # 数据优化
joblib==1.5.2 # 模型保存与加载


目标检测与裁剪：基于 YOLO11 算法精准检测并批量裁剪狗头区域，聚焦核心特征，降低背景干扰；
注意力模型构建：基于 PyTorch 搭建 ResNet50+SE 注意力机制网络，强化关键特征通道权重，提升年龄识别精度；
可视化交互：通过 PySide6 开发 UI 界面，支持图片上传、实时识别、结果展示等核心功能；
NLP 功能集成：Ollama 本地化部署 DeepSeek 大模型，基于识别的狗龄自动生成个性化护理指南；
数据处理：依托 NumPy/Pandas/Scikit-learn 完成数据预处理、数据集划分、模型性能评估。
